# VisionShield Landing Page

Uma landing page moderna desenvolvida para apresentar o **VisionShield**, um óculos inovador projetado para auxiliar na redução do risco de desenvolvimento da miopia. O projeto foi criado com foco em uma experiência intuitiva, design responsivo e comunicação clara dos benefícios do produto.

---

## Sobre o Projeto

O objetivo deste projeto é apresentar o **VisionShield** por meio de uma interface moderna e interativa, destacando suas funcionalidades, benefícios e diferenciais. A aplicação foi desenvolvida utilizando tecnologias atuais do ecossistema React, priorizando desempenho, organização de código e responsividade.

---

## Tecnologias Utilizadas

* React
* Vite
* TypeScript
* Tailwind CSS
* HTML5
* CSS3

---

## Funcionalidades

* Landing page totalmente responsiva

## Instalação

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/visionshield.git
```

Acesse a pasta do projeto:

```bash
cd visionshield
```

Instale as dependências:

```bash
npm install
```

---

## Executando o Projeto

Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

A aplicação estará disponível em:

```text
http://localhost:5173
```

---

## Gerando a Build

Para gerar a versão de produção:

```bash
npm run build
```

Para visualizar a build localmente:

```bash
npm run preview
```
